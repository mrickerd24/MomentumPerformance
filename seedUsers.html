👤 Restore Firestore User

Use this to recreate a user's Firestore document. The user must already exist in Firebase Authentication (i.e. they can still log in — they just get kicked out because their data is missing).

⚠️ You need the user's UID from Firebase Authentication console.
Go to Firebase Console → Authentication → Users, find the person, and copy their UID.
